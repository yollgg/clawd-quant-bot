# 虚拟货币量化网格交易系统 - 项目计划书

## 1. 项目概述
本系统是一个基于 Spring Boot 的轻量级虚拟货币量化交易系统。核心策略为**日内网格交易**，并结合**舆论分析**（Sentiment Analysis）来动态调整网格参数。

## 2. 核心模块
- **Binance 连接器**: 封装币安 REST API & WebSocket，获取实时行情和下单。
- **网格策略引擎**: 实现等差/等比网格，支持动态调整中轴价和区间。
- **舆论分析模块**: 监控社交媒体/新闻（模拟），判断市场波动性。
- **模拟仓模块**: 提供本地账本，记录虚拟成交、持仓和盈亏，无需真实资金即可测试。

## 3. 技术栈
- **后端**: Java 21, Spring Boot 3.4.x
- **依赖管理**: Maven
- **数据库**: H2 (内存数据库，用于模拟仓存储)
- **API 通讯**: OkHttp / Spring WebClient

## 4. 实施阶段
### 阶段一：环境搭建与 API 设计
- 初始化项目结构。
- 编写 API 接口模型。

### 阶段二：网格算法实现
- 波动率判断逻辑：如果舆论平稳且历史波动小，启动网格。
- 网格下单逻辑。

### 阶段三：模拟仓与测试
- 实现 `MockExchangeService`。
- 编写集成测试。
